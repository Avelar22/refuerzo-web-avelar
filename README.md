# Mantenimiento y Aseguramiento de la Operación de Sistemas Informáticos

## Descripción

Este proyecto consiste en una página web informativa desarrollada en HTML, CSS y Bootstrap sobre el mantenimiento y aseguramiento de la operación de sistemas informáticos.

La página tiene como objetivo explicar la importancia del mantenimiento tecnológico, los diferentes tipos de mantenimiento, herramientas utilizadas, metodologías relacionadas y los beneficios que aporta a las organizaciones.
